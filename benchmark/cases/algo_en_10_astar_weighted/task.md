# Task: Weighted A* Pathfinding

Implement a weighted A* algorithm in `astar.py`.

Requirements:
- The `grid` contains the movement cost for each cell.
- A value of `1.0` is a normal cell, while higher values mean difficult terrain.
- A value of `inf` is an obstacle.
- The path should minimize the total cumulative cost of the cells visited.
- Use Manhattan distance as the heuristic.
- Support 4-direction movement.
