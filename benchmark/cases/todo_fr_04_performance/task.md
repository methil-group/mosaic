# Tâche : Planifier l'Optimisation des Performances Web

Notre site e-commerce a un LCP (Largest Contentful Paint) de 4.5 secondes et un TBT (Total Blocking Time) trop élevé.

Votre tâche :
Rédigez un plan d'optimisation dans `performance.md`.

Exigences :
1. Proposez des solutions pour l'optimisation des images (Formats, Lazy loading).
2. Expliquez comment réduire la taille des bundles JavaScript (Code splitting, Tree shaking).
3. Mentionnez l'utilisation du cache (CDN, Service Workers).
4. Listez les outils de mesure à utiliser (Lighthouse, Web Vitals).
