# Tâche : Recherche d'Instructions Cachées

Le fichier `projet.txt` à la racine contient des informations par défaut. Des instructions spécifiques ont été stockées dans un fichier nommé `instructions.txt` quelque part dans le dossier `systeme/`.

Votre tâche :
1. Trouvez le fichier `instructions.txt`.
2. Lisez son contenu.
3. Appliquez les changements demandés (Clé et Statut) au fichier `projet.txt`.

Utilisez vos outils de navigation et de lecture pour accomplir cette tâche.
