# Tâche : Implémenter un générateur de donjon procédural

Dans `dungeon.js`, implémentez un algorithme simple pour générer une carte de donjon.

## Exigences :
1. **Algorithme** : Utilisez une stratégie de "Marche Aléatoire" (Random Walk) ou de placement de pièces.
2. **Sortie** : La grille doit être remplie de `.` pour le sol et `#` pour les murs.
3. **Connectivité** : Au moins 30% de la grille doit être de l'espace au sol.
4. **Variété** : La sortie ne doit pas être une carte statique prédéfinie ; elle doit être générée de manière procédurale.
