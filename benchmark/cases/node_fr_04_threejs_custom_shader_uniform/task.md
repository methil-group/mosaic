# Tâche : Shaders Personnalisés et Uniforms

Dans `app.js`, implémentez un `ShaderMaterial` qui utilise un uniform `uTime` pour animer les couleurs.

Exigences :
- Instanciez `THREE.ShaderMaterial` avec les `vertexShader` et `fragmentShader` fournis.
- Déclarez un uniform `uTime` dans l'objet `uniforms`.
- Créez une fonction ou une méthode pour mettre à jour la valeur de `uTime` à chaque frame.
