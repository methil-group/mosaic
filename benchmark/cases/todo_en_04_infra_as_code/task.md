We need to deploy a high-availability web service on AWS including a Load Balancer, an Auto Scaling Group, and a managed Database (RDS).

Your task:
Write an infrastructure plan in `infra.md`.

Requirements:

3. Explain how "State" is managed in Terraform (e.g., S3 backend).
4. List the variables needed for customization.
