# Task: Plan REST to GraphQL Migration

We are migrating our current REST API (Endpoints like `/users`, `/posts`, `/comments`) to a single GraphQL endpoint.

Your task:
Write a migration plan in `migration.md`.

Requirements:
1. Define the core Types (User, Post, Comment).
2. List the necessary Queries and Mutations.
3. Describe how to handle authentication.
4. Mention a strategy for "Incremental Migration" (e.g., using a Gateway or Proxy).
