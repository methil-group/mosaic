# Tâche : Planifier un Système de Panier d'Achat

Nous devons ajouter une fonctionnalité de panier d'achat à notre site e-commerce.

Votre tâche :
Créez un fichier nommé `todo.md` contenant le plan d'implémentation.

Exigences :
1. Listez les actions nécessaires (Ajout, Retrait, Mise à jour de quantité, Vider).
2. Expliquez comment gérer la persistance (Session, LocalStorage ou Base de données).
3. Décrivez le calcul des totaux (HT, TVA, Frais de port).
4. Listez les composants UI à créer.
