# Tâche : Planifier la Migration vers la V3

Notre application passe d'une architecture monolithique à des microservices (V3).

Votre tâche :
Rédigez un plan de migration dans `migration.md`.

Exigences :
1. Définissez les premiers services à extraire (ex: Auth, Commande).
2. Expliquez comment gérer la communication entre services (ex: RabbitMQ, gRPC).
3. Détaillez une stratégie de déploiement progressif (ex: Canary, Blue-Green).
4. Listez les nouveaux outils de monitoring nécessaires.
