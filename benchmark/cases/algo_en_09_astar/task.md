# Task: A* Pathfinding (Basic)

Implement the A* algorithm in `astar.js` to find the shortest path between a start and end point on a 2D grid.

Requirements:
- The grid contains `0` (walkable) and `1` (obstacle).
- Return an array of coordinates `[row, col]` from start to end.
- Use the Manhattan distance as the heuristic.
- Support 4-directional movement (Up, Down, Left, Right).
