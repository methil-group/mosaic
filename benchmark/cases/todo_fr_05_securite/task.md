# Tâche : Planifier un Audit de Sécurité et une Mise à Niveau

Notre application a été signalée pour des vulnérabilités potentielles (OWASP Top 10).

Votre tâche :
Rédigez un plan d'audit et de sécurisation dans `securite.md`.

Exigences :
1. Détaillez les étapes pour vérifier les injections SQL et les failles XSS.
2. Expliquez comment sécuriser l'authentification (MFA, Politiques de mot de passe).
3. Mentionnez des outils d'analyse statique et dynamique (SAST/DAST).
4. Proposez une stratégie pour la gestion sécurisée des secrets (ex: Vault, KMS).
