# Task: Plan Database Query Optimization

Our application is experiencing slow load times on the user dashboard. The main query fetching user activities, notifications, and profile data from a SQL database is taking over 2 seconds.

Your task:
Write an optimization plan in `optimization.md`.

Requirements:
1. Suggest techniques for diagnostic (e.g., EXPLAIN ANALYZE).
2. Propose indexing strategies (e.g., B-Tree, Compound indexes).
3. Mention caching layers (e.g., Redis).
4. Discuss potential query refactoring (e.g., avoiding N+1 problems).
