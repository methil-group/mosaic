# Task: Implement a Procedural Dungeon Generator

In `dungeon.js`, implement a simple algorithm to generate a dungeon map.

## Requirements:
1. **Algorithm**: Use a "Random Walk" or "Room Placement" strategy.
2. **Output**: The `grid` should be filled with `.` for floor and `#` for walls.
3. **Connectivity**: At least 30% of the grid should be floor space, and all floor tiles should ideally be connected (though a simple floor density check will be used for verification).
4. **Variety**: The output should not be a static predefined map; it must be procedurally generated.
