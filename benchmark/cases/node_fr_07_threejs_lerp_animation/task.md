# Tâche : Animation par Interpolation (Lerp)

Dans `app.js`, animez le cube pour qu'il se déplace de façon fluide entre deux points en utilisant l'interpolation linéaire.

Exigences :
- Utilisez `cube.position.copy(startPos)`.
- Appelez `cube.position.lerp(endPos, alpha)`.
- Gérez la progression de `alpha` entre 0 et 1.
