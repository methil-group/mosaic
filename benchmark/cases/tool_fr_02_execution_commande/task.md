# Tâche : Génération d'Inventaire via Ligne de Commande

Votre tâche est de générer une liste récursive de tous les dossiers et fichiers présents dans le workspace (sauf les dossiers système cachés).

Exigences :
1. Exécutez une commande comme `ls -R` ou `find .` pour obtenir la liste complète des fichiers.
2. Enregistrez le résultat dans le fichier `inventaire.txt`.
3. Le fichier doit contenir au moins les noms de dossiers trouvés.
