# Tâche : Chargement et Configuration de Texture

Dans `app.js`, utilisez `TextureLoader` pour charger une image et l'appliquer correctement à la sphère.

Exigences :
- Chargez la texture `'assets/earth.jpg'`.
- Appliquez-la au `material.map`.
- Réglez `wrapS` sur `THREE.RepeatWrapping`.
- Réglez `repeat.x` à `2`.
